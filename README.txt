xSchedule is a show player/scheduler designed to control
USB/DMX/sACN(e1.31)/ArtNET(e.1.17)/DDP controllers.


Latest releases are found at https://github.com/xLightsSequencer/xSchedule
Videos tutorials are found at videos.xlights.org
Issue Tracker is found here: https://github.com/xLightsSequencer/xSchedule/issues
